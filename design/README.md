# Бренд Stenogram

Исходные дизайн-материалы приложения (переименовано из VoxMate 2026-07-17).

- **Название:** Stenogram
- **Слоган:** Мобильный AI-ассистент
- **Бренд-цвет (бирюзовый):** `#2FB3B7`
- **Иконка:** белый микрофон со строками-транскриптом в скруглённом бирюзовом квадрате.

## Файлы

- `0d42ad6b-…jpg` — чистая иконка приложения (источник для растеризации).
- `32b200ff-…jpg`, `160d6789-…jpg` — логотип с подписью на разных фонах.
- `39906533-…jpg`, `752e7de6-…jpg` — варианты лого/иконки на тёмном и бирюзовом фоне.
- `ed3f1baf-…jpg` — визитки.
- `photo_5350831989105302746_y.jpg` — промо-фото.

## Рабочие иконки

Продакшен-иконки лежат в `frontend/public/icons/` (`icon.svg` — редактируемый исходник,
из него растрируются PNG). Манифест PWA — `frontend/vite.config.js`.
