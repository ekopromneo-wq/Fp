## 5. «Собери готовый PWA, выложи на сервер для скачивания только по паролю»

- Фронтенд был лишь назван «VoxMate PWA», но PWA не был. Добавил
  `vite-plugin-pwa`: манифест, service worker (Workbox), иконки (нарисовал из
  SVG-логотипа через Playwright).
- Собрал, упаковал в `voxmate-pwa.zip`, выложил на сервер за HTTP Basic Auth
  (Caddy `/download/*`, логин `voxmate`). Проверил: без пароля 401, с паролем —
  zip. Коммит `9f628bb`.

## 6. «Удобнее открывать/устанавливать сам PWA с сервера за паролем»

Раз PWA на том же домене, что API — всё стало same-origin (нет CORS, cookie
работает). Caddy стал отдавать распакованный билд в корне за тем же Basic Auth;
`/api` и `/health` оставил открытыми. Проверил вживую (Playwright + Basic Auth).
Коммит `0a51529`.

## 7. «Нет предложения установить приложение»

- Добавил свою кнопку «Установить приложение» (событие `beforeinstallprompt`) +
  инструкцию для iOS. Коммит `a1d635b`.
- Пользователь всё равно не увидел кнопку ни на телефоне, ни в десктопе.
  Диагностика через Chrome DevTools Protocol показала: **Basic Auth ломает
  устанавливаемость** — движок установки и service worker подгружают ресурсы
  вне активной вкладки, где пароля нет → 401 → приложение не устанавливаемо.

## 8. Решение: сменить модель доступа (выбор пользователя)

Предложил 3 варианта; пользователь выбрал **«вход по паролю аккаунта»**:

- Убрал Basic Auth с оболочки приложения — `/`, манифест, иконки, `sw.js`,
  ассеты теперь публичны и устанавливаемы. Basic Auth остался только на
  `/download/*` (zip).
- Защита доступа — вход по аккаунту; регистрация закрыта
  (`ALLOW_REGISTRATION=false`: `register` → 403, ссылка регистрации скрыта,
  флаг в `/api/auth/me`). Создан аккаунт администратора.
- `beforeinstallprompt` теперь перехватывается рано (в `index.html`).
- Проверено вживую без пароля (11/11): оболочка публична, Chrome считает
  устанавливаемым, вход/сессия работают, регистрация 403, консоль чистая, zip
  по-прежнему за паролем. Коммиты `08f2d04`, журнал `96c4e67`.

**Итог:** приложение открывается и устанавливается на `https://vox.ekoprom.org/`
(вход по аккаунту, без Basic Auth); zip — `/download/voxmate-pwa.zip` за Basic
Auth. Пароли переданы пользователю в чате, в git не коммитились.
